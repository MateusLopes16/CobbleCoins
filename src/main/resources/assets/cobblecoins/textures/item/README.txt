# Texture Files Needed

This folder should contain the following 16x16 PNG textures:

1. `cobblecoin.png` - A bronze/copper colored coin with a Pokéball symbol
2. `silver_cobblecoin.png` - A silver colored coin with a Pokéball symbol  
3. `gold_cobblecoin.png` - A gold colored coin with a Pokéball symbol

## Suggested Design

Each coin should be a simple circular coin design (16x16 pixels) with:
- A slight 3D effect (highlight on top-left, shadow on bottom-right)
- A small Pokéball or "CC" symbol in the center
- Color variations:
  - Bronze/Copper for regular CobbleCoin (#B87333)
  - Silver for Silver CobbleCoin (#C0C0C0)
  - Gold for Gold CobbleCoin (#FFD700)

You can create these textures using:
- Aseprite
- GIMP
- Piskel (free online pixel editor)
- Or any other pixel art tool

Place the finished PNG files in this directory.
